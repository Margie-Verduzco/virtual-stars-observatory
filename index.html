Virtual Stars Observatory

How to use this Observatory

✹ This visualization shows real stars from the HYG catalog.

⌕ Search bar: Find a star by name (e.g. Sirius, Vega).

🖱 Hover: Inspect star properties like magnitude, distance and color.

⊕ Ctrl + Click: Focus camera on a star.

◑ Color: Blue = hot stars, red = cool stars.

✧ Orbit: You can explore freely using mouse controls.
